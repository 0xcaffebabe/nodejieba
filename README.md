# NodeJieba

## 概述

NodeJieba只是CppJieba简单包装而成的node扩展。

具体使用看demo/segment.js 代码吧，不到5行。

还有一些废话，如果想看可以 [点我][blog1]

## 你需要安装的

* node (>=10.0 recommended)

## 运行和演示

```
node-gyp configure build && node demo/segment.js
```

## 测试

在`node v0.10.2`下测试通过
